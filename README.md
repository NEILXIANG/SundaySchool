# SundaySchool

推荐老师使用“打包版最少操作流程”：运行 `SundayPhotoOrganizer`，按窗口里显示的 `Work folder` 路径把照片放到 `input/`（通常为程序所在目录；若不可写会自动回退到桌面或主目录），再运行一次即可。
