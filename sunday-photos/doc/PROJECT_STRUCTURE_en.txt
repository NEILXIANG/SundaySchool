sunday-photos Project Structure (Console Edition)

Notes:
- Console/CLI only; no GUI.
- Input root defaults to input/ (old classroom_dir is still accepted).
- student_photos/ holds reference photos, named Name or Name_index.jpg (e.g., ZhangSan.jpg or ZhangSan_1.jpg).
- class_photos/ holds classroom photos, preferably by date folders.

Usage (source run):
1) Install deps: pip install -r requirements.txt
2) Prepare photos: reference in input/student_photos/, classroom in input/class_photos/ (date subfolders recommended)
3) Run: python run.py
4) View results in output/

Usage (packaged):
1) Deliver release_console/
2) Double-click SundayPhotoOrganizer or run launcher script
3) Place photos as prompted, rerun to finish organizing

Doc pointers:
- PROJECT_STRUCTURE.txt: Directory map and quick run steps.
- 部署说明.md / DeploymentGuide_en.md: Delivery package composition; input/output examples.
- FINAL_TEST_REPORT.md: Full pre-delivery test results.
- REPAIR_SUMMARY.md: Maintenance and fixes log.
- PRD.md: Requirements.
- TESTING.md: How to run tests.
