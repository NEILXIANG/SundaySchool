老师快速开始（macOS / Windows）

你只需要做 3 步：
1) 解压
2) 放照片到 input
3) 点运行，去 output 拿结果

第 1 步：解压
- 下载压缩包并解压到桌面（推荐）

第 2 步：放照片到 input
- input/student_photos/：放学生参考照（每个学生 1~5 张，清晰正脸最好）
- input/class_photos/：放课堂/活动照（需要整理的所有照片）

示例：
- input/student_photos/张三_1.jpg
- input/student_photos/张三_2.jpg
- input/student_photos/李四_1.jpg
- input/class_photos/2025-12-25_活动_001.jpg

第 3 步：运行（只点一个入口）
- macOS：双击 SundayPhotoOrganizer（或 启动工具.sh）
- Windows：双击 Launch_SundayPhotoOrganizer.bat
- 运行中会出现黑色窗口显示进度，请不要关闭

结果在哪里：
- output/：整理结果
- logs/：日志（出问题把 logs 整个发我）

macOS 常见提示：无法打开/无法验证开发者
- 系统设置 → 隐私与安全性 → 找到被阻止的提示 → 点“仍要打开（Open Anyway）” → 再打开一次
- 或：右键 SundayPhotoOrganizer → 打开 → 再确认一次

如果出错，请发我两样：
1) logs/ 文件夹（整个打包）
2) 报错截图
