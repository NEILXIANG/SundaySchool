üìÅ Folders created on first run (next to the app)

- input/class_photos: Classroom photos. Prefer date subfolders (2024-12-21/group.jpg). jpg/png supported.
- output: Sorted results, organized by student ‚Üí date.
- logs: Run logs; safe to clear.

üì∏ Photo prep tips

- Avoid blurry or side profiles in classroom photos when possible.

üí° Run notes

‚ùì FAQ
- Finishes too fast: Check the `Work folder` path shown in the console and verify folder names and photo placement.
- Work folder location: By default, input/output/logs are created next to the app. If that location is not writable, it falls back to Desktop (or Home). Use the printed `Work folder` path as the source of truth.
- Poor matches: Add more clear reference photos for that student.
- Won't launch (macOS): If blocked, allow "Open Anyway" in Privacy & Security.
