# SundayPhotoOrganizer（控制台版）

## 快速开始

### macOS
双击 `启动工具.command`（推荐）或 `启动工具.sh`

### Windows
双击 `Launch_SundayPhotoOrganizer.bat`

### 使用步骤
1. 学生参考照放 `input/student_photos/<学生名>/`
2. 课堂照片放 `input/class_photos/`
3. 再次运行，结果在 `output/`

## 工作目录
默认在启动器所在目录（解压后的文件夹根目录）。
如果不可写，程序自动回退到桌面/主目录，控制台会显示实际路径。
