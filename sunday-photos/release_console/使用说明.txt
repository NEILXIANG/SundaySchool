主日学照片整理工具 - 控制台版使用说明 / Sunday School Photo Organizer - Console Guide (Bilingual)

【使用方法 | How to run】
- 双击运行：双击 "SundayPhotoOrganizer"（或在终端执行 ./SundayPhotoOrganizer）
- Double-click to run: "SundayPhotoOrganizer" (or run ./SundayPhotoOrganizer in Terminal)

【快速开始 | Quick Start】
- 双击运行 "SundayPhotoOrganizer"（或终端 ./SundayPhotoOrganizer）
- 按提示将照片放入自动创建的桌面文件夹，再次运行即可整理；输出自动打开
- Double-click "SundayPhotoOrganizer" (or run ./SundayPhotoOrganizer).
- Place photos into the auto-created desktop folder, rerun to organize; output opens automatically.

【文件夹位置 | Folder location】
- 桌面/Desktop：Desktop/主日学照片整理

【自动创建的文件夹 | Auto-created folders】Desktop/主日学照片整理
- student_photos：学生照片（学生参考照），命名“姓名”或“姓名_序号”（张三.jpg、张三_2.jpg、LiSi.png）
	student_photos: reference photos, named "Name" or "Name_number" (ZhangSan.jpg, ZhangSan_2.jpg, LiSi.png)
- class_photos：课堂照片（课堂照），建议按日期子目录（2024-12-21/合影.jpg）
	class_photos: classroom photos, date subfolders recommended (2024-12-21/group.jpg)
- output：整理结果，按学生 → 日期分层
	output: sorted results by student → date
- logs：运行日志，可清空
	logs: run logs, safe to clear

【照片准备 | Photo prep】
- 每人至少 1 张清晰正脸参考照；2-3 张可提升识别率
	At least 1 clear frontal reference per student; 2–3 improves accuracy.
- 课堂照支持 jpg/png，模糊或侧脸会降低匹配效果
	Classroom photos: jpg/png; blurry/side profiles reduce matches.

【运行提示 | Run notes】
- 首次无照片会直接结束；先放照片再运行
	First run with no photos will exit; add photos then rerun.
- 可多次运行，仅整理新增课堂照；结束后自动打开 output
	Re-run anytime; only new classroom photos are organized; output auto-opens.

【重要说明 | Important note】
- 程序可能会把 class_photos 根目录照片按日期移动到 YYYY-MM-DD/ 子目录，这是正常现象
	The app may move photos into YYYY-MM-DD date folders; this is expected.

【常见问题 | FAQ】
- 很快结束 / finishes too fast：检查桌面/主日学照片整理 的目录与命名
- 识别不准 / poor matches：
	1) 为该学生多放 2-3 张清晰正脸参考照（只拍该学生一个人）
	2) 课堂照尽量光线足、人脸不要太小/太模糊
	3) 如有原图，尽量用更清晰的原图替换
- 无法启动 / won't launch：确保在 macOS 上运行，若被拦截到“隐私与安全性”→“仍要打开” (Open Anyway)
